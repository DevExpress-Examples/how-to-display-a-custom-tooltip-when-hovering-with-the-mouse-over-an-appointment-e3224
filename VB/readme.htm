Developer Express Example
How to display a custom ToolTip when hovering with the mouse over an appointment

This example illustrates how to customize an appointment ToolTip (by default, it invokes an appointment context menu). To do this, you should override a CustomAppointmentTooltip template of the ASPxScheduler (see Templates (http://documentation.devexpress.com/#AspNet/CustomDocument4550)). Basically, you can use the ASPxClientScheduler.RefreshClientAppointmentProperties Method (http://documentation.devexpress.com/#AspNet/DevExpressWebASPxSchedulerScriptsASPxClientScheduler_RefreshClientAppointmentPropertiestopic) to obtain standard and custom appointment fields (see http://www.devexpress.com/scid=Q277355). However, in v2010 vol 2 we have implemented a simpler and more flexible mechanism for preparing arbitrary values (not only standard and custom field values) for a client-side appointment object: http://www.devexpress.com/scid=S33646. In previous versions, a similar functionality can be achieved only when using custom JSProperties (see http://www.devexpress.com/scid=Q276425). Also, there is a CalculatePosition() method in the ASPxClientToolTipBase class. You can override it in the inheritor (custom tooltip) to provide custom coordinates as shown in the http://www.devexpress.com/scid=Q273610 thread.

You can find sample updates and versions for different programming languages here:
